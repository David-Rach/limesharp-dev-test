Limesharp Developer Test

To test out the functions, please open the console and enter any of the following commands there.

RepeatArray(array, count)

Array can be any valid array e.g [1,2,3]

Count is how many times to repeat it

FormatString(string)

String can by anything. Makes everything but the first letter into a lower case

GetNextBinaryNumber(array)

Array needs to follow binary convention. Ones and Zeros only e.g [1,0,0,0,1]
